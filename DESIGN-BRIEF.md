# ГЛЯНЕЦ — дизайн-бриф прототипа (nails-landing-proto)

> Сводка для проверки на другой модели/другим экспертом: что за проект, какие решения приняты и почему. 19.07.2026.

## Задача

Отклик на заказ YouDo «Сделать лендинг»: 3D-сайт записи для ногтевого салона, бюджет 30 000 ₽. Стратегия — не текстовый отклик, а готовый рабочий прототип. Параллельная цель — заготовка в личную «витрину прототипов» для похожих заказов в будущем.

## Референсы

- **Venetian Nail Spa** — атмосфера «спокойной роскоши»: светлый благородный фон под фото. Источник решения заменить тёмную тему по умолчанию на светлую золотистую.
- **Savic Motorcycles** («Configurator») — идея живого интерактивного конструктора выбора вариантов. Адаптирована в «Конструктор дизайна» (выбор формы + цвета ногтя с 3D-превью).
- Подборки лучших сайтов бьюти- и ногтевой сферы (Awwwards, Colorlib) — ориентир по уровню 3D/анимаций.

## Дизайн-система

- **Типографика:** заголовки — Bodoni MT / Didot / Big Caslon / Georgia (serif); текст — Century Gothic / Futura / Trebuchet MS / Segoe UI (sans-serif).
- **Палитра:** светлая кремово-золотистая по умолчанию (`--bg:#f6ede2`, акцент `--wine:#6d2331`, `--rose:#c98a6b`). Тёмная тема — опция по тумблеру в шапке, не дефолт.
- **3D:** hero-карусель лаков (CSS 3D-transform + parallax от курсора), флип-карточки услуг, 3-шаговая форма записи с 3D-флипом между шагами, «Конструктор дизайна» с живым 3D-превью формы/цвета ногтя (интегрирован с формой записи).
- **Hero-фон:** AI-сгенерированный натюрморт (флаконы, кисть, мрамор) — без людей и рук. Первая версия с руками имела артефакт (двоящаяся рука в углу), после чего решили полностью убрать людей из кадра и внимательнее проверять AI-фото на анатомические дефекты перед использованием.
- **Бренд:** «ГЛЯНЕЦ» — крупный заголовок прямо в hero (после критики, что раньше было мелким логотипом в шапке, непонятным при первом взгляде).
- **Футер:** контакты + мессенджеры (Telegram/WhatsApp/VK, inline SVG-иконки), блок из 4 юридических документов (Политика конфиденциальности / Согласие на обработку ПДн / Публичная оферта / Политика cookie — каждый отдельной страницей в `legal/`), реквизиты ООО (все данные вымышленные, явно помечено в футере), cookie-баннер с localStorage.

## Технический стек

- Один самодостаточный `index.html` (без сборки, без внешних запросов) + синхронная копия `artifact.html` (без DOCTYPE/head/body — формат для Claude Artifact) + 4 страницы юр.документов в `legal/`.
- Хостинг: GitHub Pages — github.com/Allusa394/nails-landing-proto → **https://allusa394.github.io/nails-landing-proto/** (обновляется автоматически по `git push` в main).
- Резервная ссылка: Claude Artifact **https://claude.ai/code/artifact/67003960-8839-4eec-b9bd-5f03138e0816** (приватная, пока не нажать «Поделиться»).

## Проверено (Puppeteer, на каждом шаге)

Десктоп (1440×900, 1920×1000) и мобильный (390×844) — без горизонтального переполнения, фокус-состояния для клавиатуры, `prefers-reduced-motion` учтён, все 4 юр.страницы открываются и содержат рабочую ссылку «назад на сайт».

## Проверка на другой модели — выполнена 19.07.2026

QA-прогон (Puppeteer, 30 сценариев) + дизайн-ревью + сравнение с эталонами (Colorlib 28 best 2026, GlossGenius). Найдено и починено:
- валидация телефона (принимал «абвгд»), тап-таргеты до 44px, favicon (был 404), мобильное меню (пункт «Запись» прятался за краем), тёмная тема hero (scrim через CSS-переменную `--hero-scrim` + светлый текст), угловатые формы ногтей в конструкторе (плавные скругления).
- «Работы»: CSS-градиенты заменены на 6 реальных фото маникюра (Pexels/Unsplash, свободная лицензия, кредит в футере) — файлы в `img/`, в artifact.html встроены base64 (скрипт пересборки: scratchpad/sync-artifact.js — регенерирует artifact.html из index.html).

## Известные открытые пункты (не критично)

1. Список цветов лака дублируется между разделами «Работы» и «Конструктор» — предложено вынести в общий JS-массив, не подтверждено.
2. В сетке «Работ» на широких экранах остаются пустые ячейки под короткими карточками (было так же с градиентами) — можно перебалансировать tall-карточки.
3. Карточка 3D-карусели в hero, повёрнутая ребром, видна как тонкая вертикальная линия — мелкий артефакт.

## Вопросы для проверки на другой модели

1. Соответствует ли уровень 3D-исполнения лучшим сайтам бьюти-ниши (не примитивно ли)?
2. Читаемость, контраст и типографическая иерархия текста на светлом фоне.
3. Структура футера — покрывает ли она требования 152-ФЗ по составу документов и реквизитов.
4. Мобильная адаптивность и скорость восприятия на маленьком экране.
